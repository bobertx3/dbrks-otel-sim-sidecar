Event Stream Legend

Priority Levels

P1
Critical — Full outage, major revenue impact, SLA breach
e.g., "Payment Gateway Down"
P2
High — Significant degradation, large user impact
e.g., "SAP Batch Queue Delay"
P3
Medium — Partial issues, moderate impact
e.g., "Order API Slow Response"
P4
Low — Normal operations, no impact
e.g., "Order API Request" — healthy baseline

Severity Indicators

Normal — Healthy baseline traffic, no action needed
Warning — Degraded performance, fires app + infra layer events
Critical — Full cascade across all 3 layers (app → infra → network)

SLA Badge

SLA
SLA Breach — Incident violates Service Level Agreement thresholds (e.g., 99.9% uptime, MTTR under 60 min). Only critical P1 scenarios trigger SLA breaches. Counts toward the "SLA Breaches" KPI on the executive dashboard.

Triplet Tags

order-mgmt-chain
Shows which application triplet (App → Infra → Network chain) the event belongs to. Each triplet maps one application to its infrastructure pod and network subnet.
